# Words to Avoid (Internal Reference)

**Not for public site.** Use plain-language alternatives on all customer-facing pages.

| Avoid | Use instead |
|-------|-------------|
| Foundation model | Decision software / decision layer |
| Calibrated confidence | Knows how sure it is |
| Runtime safety contract | Safety rule built into every decision |
| Abstain | Wait / don't act |
| Escalate | Call a human |
| Physics-grounded | Based on how the real system actually works |
| Multiphysics correlation | Matches real-world behavior in tests |
| Novelty vs conflict | New situation vs genuinely unclear |
| QPU / qubits (in hero) | Advanced hardware research (footer only) |

## Additional guidance

- **Commit / hold** → Act / wait / call a human
- **Unsafe commitments** → Unsafe automatic actions
- **Perceive, design, decide, speak, create** → Avoid on marketing pages; describe outcomes instead
- **Bolt-on filter / substrate** → Decision layer that sits under what you already run
